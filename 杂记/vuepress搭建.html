# 使用 vuepress 搭建博客

引言：因为之前使用 hexo 感觉搭出来的博客太过花里胡哨，所以重新使用 vuepress 进行搭建

官网：https://www.vuepress.cn

需要准备 node.js, webstorm

# 创建nodejs项目

将如下的srcipt脚本替换生成的node项目中的那部分

{
  "scripts": {
    "dev": "vuepress dev docs",
    "build": "vuepress build docs"
  }
}

如果没有任何配置，这个网站将会是非常局限的，用户也无法在你的网站上自由导航。为了更好地自定义你的网站，让我们首先在你的文档目录下创建一个 .vuepress 目录，所有 VuePress 相关的文件都将会被放在这里。你的项目结构可能是这样：

参照官网-基本配置

.
├─ docs
│  ├─ README.md
│  └─ .vuepress
│     └─ config.js
└─ package.json

一个 VuePress 网站必要的配置文件是 .vuepress/config.js，它应该导出一个 JavaScript 对象：

module.exports = {
    title: 'winden96',
    description: 'winden96 的学习记录'
}

再启动 dev，即可开启我们的博客之旅了

# 自动生成侧边栏

参考：https://github.com/shanyuhai123/vuepress-plugin-auto-sidebar

然后我又把之前的首页删了，没要首页了

# 介绍（Introduction）

这是为 vuepress 自动生成侧边栏的插件。 This is a plugin that automatically generates a sidebar for vuepress.

# 安装（Install）

# 1. VuePress v1

npm i vuepress-plugin-auto-sidebar -D

# 2. VuePress v2

# vuepress v2 alpha(vuepress v2 仍处于测试阶段)
npm i vuepress-plugin-auto-sidebar@alpha -D

# 使用（Usage）

📖 更详细的文档见 vuepress-plugin-auto-sidebar (opens new window)。 📖 For more detailed documentation, you can visit vuepress-plugin-auto-sidebar (opens new window).

VuePress v1：

// 在 .vuepress/config.js 中配置插件
// edit .vuepress/config.js file
module.exports = {
  plugins: [
    ["vuepress-plugin-auto-sidebar", {}]
  ]
}

VuePress v2：

VuePress v2 (opens new window) 不再支持插件修改 sidebar，所以你需要自行引入生成的 sidebar.js 文件。 VuePress v2 (opens new window) no longer supports plugins to modify sidebar, so you need to import the generated sidebar.js file yourself.

const sidebarConf = require('./sidebar')

module.exports = {
  plugins: [
    ["vuepress-plugin-auto-sidebar", {}]
  ],
  themeConfig: {
    sidebar: sidebarConf
  }
}
