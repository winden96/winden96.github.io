# 初识 Hexo

# 1. 快速搭建自己的博客

首先需要安装 node.js，安装 npm 模块，切换为淘宝镜像源，更多可在其他文章下查看

本人使用的是win10系统进行搭建

具体文档：https://hexo.io/zh-cn/docs/

参照视频：https://www.bilibili.com/video/BV1Yb411a7ty?spm_id_from=333.999.0.0

npm install -g hexo-cli

然后便可通过 hexo -v 查看信息

生成一个文件夹 mkdir block

然后进入该文件夹下 cd block

进行博客初始化 hexo init，会从 github下clone，所以会有点慢

下载完成后，可通过 dir 进行查看目录信息，linux通过 ls -l查看

然后可通过 hexo s 这里 s 为 start 的简写，进行启动，浏览器输入 http://localhost:4000/便可查看

通过 hexo n "我的第一篇博客文章" 便可创建

然后通过 hexo clean清除缓存 网页正常情况下可以忽略此条命令，所以说这里可以不用

通过 hexo g 进行生成静态网页

再次hexo s 启动, 便可在 本地 看到变化 http://localhost:4000/

# 2. 放入github进行外网访问

  • 首先安装 git模块，当然也可以通过 git bash 直接进行
npm install --save hexo-deployer-git
  • 在 github/gitee 上面 新建一个仓库进行管理，这里采用 github（先试试水）

  • 在 你所创建的 hexo 目录下有个 _config.yml，Deployment 修改其中内容
# Deployment
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
  type: git
  repo: https://github.com/winden96/winden96.github.io.git
  branch: main

这里我的github账号默认是 main 分支，也可以新建一个 master 分支，然后通过 git checkout -b master 进行新建并切换到 master 分支

  • 通过 hexo d 进行部署 等价 (hexo deploy)

然后就可以通过 https://winden96.github.io/ 进行访问了

# 3. 修改默认主题

选择 https://github.com/litten/hexo-theme-yilia.git 这个主题

  • 克隆到本地 的 themes/yilia 目录下，因为之前不是安装了 git模块吗，可以cmd直接克隆
git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

  • 还是修改之前的那个 _config.yml 文件
# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/
theme: yilia

将主题改成 yilia

重新部署便可以得到了，依次执行

hexo clean
hexo g
hexo d
